rm -rf script.py (löscht datei)

nano script.py (nano ist bearbeiten strg X raus) STRG O zum speichern
./ zum ausführen von Dateien

python 3 script.py

Tipp 5

SSH-Verbindung

ifconfig
sudo service ssh status
sudo apt-get update
sudo apt-get remove openssh-server
sudo apt-get install openssh-server
sudo nano /etc/ssh/sshd_config (password authentifikation and AllowUsers alucard)
sudo ssh-keygen -A
sudo service ssh start
sudo service ssh status
sudo service ssh --full-restart

Tipp 6

Pipe

ifconfig | grep inet (Filtere ifconfig nachinet | = pipe)
cat datei1 datei2 > newfile.txt (Konkatenieren, Verbinden, bei nur einer Datei wird der Inhalt angezeigt)
sudo lsof -u alucard (list of all files)
sudo lsof -u alucard | more (in Tabellenstruktur)
sudo kill (prozessid)
sed 's/Linux/Unix/g' readme.txt (ändere alle Linus in Unix um in readme(temporär))
außer vorher sudo chmod 777 readme.txt
sed -i 's/Linux/Unix/g' readme.txt
sed --help für infos über die Funktion(/? oder -help auch möglich)

Tipp 7

awk (formatierung)

cat colums.txt | awk 'geschweifte(print $1)' (erste Zeile von Tabelle)
cat colums.txt | awk 'print $NF'(NF = das allerletzte in der Tabelle)
