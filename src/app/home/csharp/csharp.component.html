Tipp 9

LINQ Language Integrated Query

bisschen wie SQL
Die Frage: IEnumerable<string> scoreQuery =
from score in scores //required
where score > 80
orderby score descending
select $"The score is {score} //select or group
Die Antwort: foreach(string s in scoreQuery){
Console.WriteLine(s); }

Tipp 10

Alternativ LINQ with methods

var scoreQuery = scores.Where(s => s > 80)
.OrderByDescending(s => s);
select is implied, ähnlich wie Python oder JS
