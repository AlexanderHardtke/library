HTML Tipps to copy paste

Tipp 1

HTTP Methoden

GET zum bekommen von HTML-Datei oder einem JSON-Objekt vom Browser
POST zum hinzufügen eines JSON (id, item, etc), gibt Antwort zurück.(Nummer 200 Ok oder 404 not found), Ladebalken währenddessen einblenden

Tipp 2

HTTP Methoden 2

PUT möchte ein JSON verändern z.B die Anzahl.
DELETE entfernt das JSON-Objekt

Tipp 3

Einloggen

mit POST-Methode, email + PW, => JSON true, sessionID(hash) wird hinzugefügt(lokal Storage oder Cookie)
bei darauffolgender anmeldung GET mit URL + SessionID

Tipp 4

Einhacken mit SessionID

GET mit tausenden SessionID's (mehr Zeichen = mehr Sicherheit), (begrenzt gültig 15min, z.B. Bank), (WebbrowserFingerprint(ID) "+ Bestätigung"), 2FA mit Handy, etc

Tipp 5

Load Balancer

Ist zwischen FTP-Server und Browser geschaltet, sucht sich einen von mehreren Servern aus(Facebook), damit Server nicht überlastet wird. Wird umgeleitet bei Ausfall auf andere Server

Tipp 6

Travis UI (nur GitHub), Jenkins (Bitbucket), GitLabCItools (Gitlab)

Ist ein Service der automatisch auf git master pushed und direkt auf FTP-Server den Content aktualisiert. Kann auch vorher den Code ng build --prod, dann testen und dann erst hochladen

Tipp 7

Hoisting-Prinzip

JS shuffled automatisch während Python nicht auf Funktionen zugreifen kann die noch nicht definiert sind. Mithilfe des Compilers

Tipp 8

XSS/CrossSiteScripting

wenn JS Text schreibt sollte man > < verwenden und kleiner als als Zeichen blockieren/ersetzen

Tipp 11

p vs span

p wird als neue Zeile verwender also ein Absatz in einem Text
Ein span ist innerhalb der Line um zum Beispiel einen Text zu ändern
