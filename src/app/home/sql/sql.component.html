Tipp 1

Grundlagen SQL

SQL ist eine Tabellendatenbank mit einfachen Befehlen die man aber miteinander kombinieren kann

Tipp 2

Zeige alle Tabellen

SELECT name
FROM sqlite_master
WHERE type = 'table'

Tipp 3

Tabelle anzeigen

SELECT * FROM Orders
Tabelle limitieren und nach bestimmtem Kriterium filtern
ASC oder DESC oder Limit setzen
ORDER BY Quantity LIMIT 2

Tipp 4

Reihe hinzufügen

kann Reihen NICHT austauschen
INSERT INTO Customers(CustomerID, CustomerName, ContactName, Country)
VALUES ("6" ,"Alex the keks", "AlexKeks", "Germany")

Tipp 5

Filtern

Filtert nach einem oder mehreren Eingaben
SELECT * FROM Orders WHERE ProductID=2
SELECT * FROM Orders WHERE ProductID IN (1,2)

Tipp 6

Erweitertes Filtern

mit Filtern eines Strings mit Like "%" SELECT * FROM Customers WHERE (ContactName LIKE "A%" AND Country="Germany") OR (Country="Sweden" AND CustomerName LIKE "B%")

Tipp 7

Tabellen verbinden

SELECT * FROM Orders
LEFT JOIN Customers
ON Orders.CustomerID = Customers.CustomerID
WHERE Country="Mexico"

Tipp 8

Join mit neuer Tabelle

CREATE TABLE ProductOrders AS SELECT ProductName, CustomerName, Price, Quantity FROM Products
LEFT JOIN Orders
ON Orders.ProductID = Products.ProductID
LEFT JOIN Customers
ON Orders.CustomerID = Customers.CustomerID
WHERE ProductName = "Chais"

Tipp 9

Wert in Tabelle löschen

DELETE FROM ProductOrders WHERE "Chais"

Tipp 10

Tabelle leeren

DELETE FROM ProductOrders

Tabelle löschen
DROP TABLE ProductOrders

Tipp 11

Rechnen in neuer Spalte

SELECT ProductName, CustomerName, Price, Quantity, Price * Quantity AS Total FROM ProductOrders

Tipp 12

Summe und Average

SELECT SUM(Price * Quantity) AS Total, AVG(Price * Quantity) AS Average FROM ProductOrders

Tipp 13

Daten Updaten

Tipp 14

SQL Django

in SQL:
SELECT*FROM employees
WHERE Alter < 30 AND ...
 in Django:
employees.filter(alter__lte=30)
in Django einfacher zu schreiben als reines SQL
